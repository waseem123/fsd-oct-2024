Tags and Attributes

Tags are the basic building blocks of any HTML web page structure. Tags are written using angular brackets. and there are multiple predefined tags present in HTML.
The tags will decide that which type of representation the conent wll be given.

Attributes are used as specific properties given on specific tag.
